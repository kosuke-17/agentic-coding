# ドキュメント生成

handson/src/ のコードから API ドキュメントを生成します。

## 対象

- handson/src/*.ts

## 出力形式

各関数について以下の情報を Markdown で出力してください:

---

## 関数名

### 説明

関数の目的と動作を簡潔に説明

### シグネチャ

```typescript
function 関数名(パラメータ): 戻り値の型
```

### パラメータ

| 名前 | 型 | 必須 | 説明 |
|------|---|------|------|
| param1 | Type | ✅/❌ | 説明 |

### 戻り値

| 型 | 説明 |
|---|------|
| ReturnType | 説明 |

### 例外

| 条件 | エラーメッセージ |
|------|----------------|
| 条件の説明 | "エラーメッセージ" |

### 使用例

```typescript
// 正常系の例
const result = 関数名(引数);
console.log(result); // 期待される出力

// 異常系の例
try {
  関数名(不正な引数);
} catch (error) {
  console.error(error.message);
}
```

### 注意事項

- イミュータブルな実装かどうか
- パフォーマンスに関する注意
- その他の注意点

---

## 出力先

handson/docs/api.md として保存してください。

## 追加タスク

1. ドキュメントを生成
2. handson/docs/ ディレクトリがなければ作成
3. api.md を作成/更新
4. 目次を自動生成
