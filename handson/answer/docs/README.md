# Answer: TODOアプリの解答例

## 概要

このディレクトリには、TODOアプリの完成版実装が含まれています。

## ディレクトリ構造

```
answer/
├── src/
│   ├── logic/        # 完成したロジック実装
│   └── ui/          # 完成したUIコンポーネント
├── test/
│   ├── logic/       # 完成したロジックテスト
│   └── ui/          # 完成したUIテスト
└── docs/            # ドキュメント
```

## 実装のポイント

### ロジック実装

- **純粋関数**: 全ての関数は副作用を持たない純粋関数として実装
- **イミュータブル**: 元の配列を変更せず、新しい配列を返す
- **エラーハンドリング**: 不正な入力に対して適切なエラーをスロー

### UI実装

- **カスタムフック**: `useTodos`で状態管理を集約
- **コンポーネント分割**: 責務ごとにコンポーネントを分割
- **アクセシビリティ**: aria-label等でアクセシビリティを配慮
- **スタイリング**: インラインスタイルでシンプルなデザイン

## テスト

全てのロジックと主要なUIコンポーネントに対してテストを実装しています。

### 実行方法

```bash
# answerのテストのみ実行
pnpm test answer/test/

# 特定のテストファイルを実行
pnpm test answer/test/logic/todo.test.ts
```
