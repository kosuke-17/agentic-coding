# 🎯 Cursor Agentic Coding ハンズオン

## 概要

このハンズオンでは、**TDD（テスト駆動開発）を使った TODO アプリ開発**を通じて、Cursor の強力な AI エージェント機能を体験します。

## 学習目標

このハンズオンを終えると、以下のことができるようになります：

1. **Agent Skills** - Cursor エージェントにタスクを効果的に依頼する方法
2. **Subagents** - 複雑なタスクを分担させる方法
3. **Rules** - プロジェクト固有のルールを設定してエージェントの振る舞いを制御する方法
4. **Commands** - 繰り返し使うワークフローをコマンド化する方法

## ハンズオンの流れ

```
┌─────────────────────────────────────────────────────────────────┐
│  Step 1: 環境セットアップ                                        │
│    └── Cursor プロジェクト初期化、TDD 環境構築                    │
├─────────────────────────────────────────────────────────────────┤
│  Step 2: Agent Skills で TODO アプリを開発                       │
│    └── エージェントにテスト作成・実装を依頼                       │
├─────────────────────────────────────────────────────────────────┤
│  Step 3: Subagents でタスクを分担                                │
│    └── 複雑な分析・リサーチを Subagent に委譲                     │
├─────────────────────────────────────────────────────────────────┤
│  Step 4: Rules でエージェントの振る舞いを制御                     │
│    └── TDD ルールを設定して品質を担保                            │
├─────────────────────────────────────────────────────────────────┤
│  Step 5: Commands でワークフローを自動化                          │
│    └── よく使う操作をコマンド化                                   │
└─────────────────────────────────────────────────────────────────┘
```

## 前提条件

- **Cursor** がインストールされていること
- **Node.js** (v18 以上) がインストールされていること
- **Git** の基本操作ができること
- JavaScript/TypeScript の基本的な知識

## このハンズオンで作るもの

シンプルな **TODO アプリ**を TDD で実装します：

- TODO の追加
- TODO の完了/未完了の切り替え
- TODO の削除
- TODO のフィルタリング（全て / 未完了 / 完了済み）
- ReactコンポーネントによるUI実装
- カスタムフックによる状態管理
- UIテストの実装

## ディレクトリ構成

このハンズオンには2つのディレクトリがあります：

- **handson/challenge/**: ロジック + UIの課題（学習者が実装）
- **handson/answer/**: 完成版の解答例

## 所要時間

約 **2〜3 時間**（進め方により前後します）
